# Self Test Workspace

This repository currently uses the `CreatePlttServer` app.

**Prerequisites**
1. Install Node.js (LTS recommended).
2. Ensure `npm` is available in your PATH.

**How to Run (after checkout)**
1. Open a terminal at the repo root.
2. Install dependencies:

```bash
cd CreatePlttServer
npm install
```

3. Start the dev server:

```bash
npm run dev
```

4. Open the printed local URL in your browser.

**Optional: Skip to the last step**
- Append `?skip=last` to the URL.
  - Example: `http://localhost:5173/?skip=last`

**Notes**
- `node_modules/` is intentionally ignored by Git. After checkout, always run `npm install` in `CreatePlttServer`.

# Self Test Workspace (한국어)

이 저장소는 현재 `CreatePlttServer` 앱을 사용합니다.

**사전 준비**
1. Node.js (LTS 권장)을 설치하세요.
2. `npm`이 PATH에 포함되어 있는지 확인하세요.

**체크아웃 후 실행 방법**
1. 저장소 루트에서 터미널을 엽니다.
2. 의존성 설치:

```bash
cd CreatePlttServer
npm install
```

3. 개발 서버 실행:

```bash
npm run dev
```

4. 터미널에 출력된 로컬 URL을 브라우저에서 여세요.

**선택: 마지막 단계로 바로 이동**
- URL 뒤에 `?skip=last`를 붙이세요.
  - 예: `http://localhost:5173/?skip=last`

**참고**
- `node_modules/`는 Git에서 제외합니다. 체크아웃 후에는 항상 `CreatePlttServer`에서 `npm install`을 실행하세요.
